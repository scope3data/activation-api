## Overview

The `analyze_tactics` tool provides deep, ML-powered analysis of campaign tactic performance. Get statistical insights, signal/story effectiveness rankings, attribution analysis, and actionable optimization recommendations.

## Parameters

<ParamField path="campaignId" type="string" required>
  Campaign ID to analyze
</ParamField>

<ParamField path="analysisType" type="string" required>
  Type of analysis to perform.
  
  **Options:**
  - `efficiency` - Overall tactic efficiency rankings and performance comparison
  - `attribution` - Multi-touch attribution analysis with conversion path insights  
  - `optimization` - ML-powered optimization recommendations with A/B test results
  - `signals` - Deep dive into audience signal effectiveness and statistical significance
  - `stories` - Narrative context performance analysis and engagement patterns
</ParamField>

<ParamField path="compareSignals" type="boolean" optional default="false">
  Include cross-signal performance comparison and effectiveness rankings
</ParamField>

<ParamField path="compareStories" type="boolean" optional default="false">
  Include narrative context comparison and engagement analysis
</ParamField>

<ParamField path="timeframe" type="string" optional default="7d">
  Analysis timeframe.
  
  **Options:**
  - `7d` - Last 7 days
  - `14d` - Last 14 days  
  - `30d` - Last 30 days
  - `custom` - Use customDateRange parameter
</ParamField>

<ParamField path="customDateRange" type="object" optional>
  Custom date range when timeframe is 'custom'.
  
  <Expandable title="customDateRange properties">
    <ParamField path="start" type="string" required>
      Start date in YYYY-MM-DD format
    </ParamField>
    <ParamField path="end" type="string" required>
      End date in YYYY-MM-DD format
    </ParamField>
  </Expandable>
</ParamField>

## Response

<ResponseField name="analysisId" type="string">
  Unique identifier for this analysis
</ResponseField>

<ResponseField name="campaignId" type="string">
  Campaign being analyzed
</ResponseField>

<ResponseField name="analysisType" type="string">
  Type of analysis performed
</ResponseField>

<ResponseField name="dateRange" type="object">
  Date range analyzed
</ResponseField>

<ResponseField name="tacticRankings" type="array">
  Tactic performance rankings with efficiency scores and recommendations
</ResponseField>

<ResponseField name="signalEffectiveness" type="array" optional>
  Signal performance analysis with statistical significance testing (when compareSignals=true)
</ResponseField>

<ResponseField name="storyEffectiveness" type="array" optional>
  Story performance analysis with engagement metrics (when compareStories=true)
</ResponseField>

<ResponseField name="recommendations" type="array">
  ML-powered optimization recommendations with expected impact
</ResponseField>

<ResponseField name="statisticalSignificance" type="object" optional>
  A/B test results and confidence intervals for optimization recommendations
</ResponseField>

<ResponseField name="insights" type="array">
  Key insights and pattern discoveries from the analysis
</ResponseField>

## Analysis Types

### Efficiency Analysis

**Best for:** Campaign optimization and budget reallocation

**Provides:**
- Tactic performance rankings with efficiency scores
- CPM, CTR, and conversion rate comparisons
- Signal and story effectiveness breakdown
- Budget reallocation recommendations

**Use when:** You want to optimize current campaign performance and identify top-performing tactics.

### Attribution Analysis  

**Best for:** Understanding customer journey and touchpoint effectiveness

**Provides:**
- Multi-touch conversion path analysis
- First-touch, last-touch, and assisted conversions
- Signal attribution across the customer journey
- Creative touchpoint effectiveness

**Use when:** You need to understand how customers convert and optimize touchpoint sequence.

### Optimization Analysis

**Best for:** Strategic campaign improvements with ML insights

**Provides:**
- ML-powered optimization recommendations
- A/B test insights and statistical significance
- Risk assessment for proposed changes
- Implementation roadmaps for improvements

**Use when:** You want data-driven recommendations for improving campaign performance.

### Signal Analysis

**Best for:** Audience targeting optimization

**Provides:**
- Statistical significance testing for audience segments
- Signal effectiveness rankings with confidence intervals
- Cross-signal performance comparison
- Audience expansion recommendations

**Use when:** You need to optimize audience targeting and understand segment performance.

### Story Analysis

**Best for:** Creative messaging and narrative optimization  

**Provides:**
- Narrative context performance analysis
- Engagement pattern identification
- Story effectiveness across different audiences
- Creative messaging recommendations

**Use when:** You want to optimize creative messaging and narrative approach.

## Statistical Methods

### Significance Testing

All analyses include statistical significance testing:

- **T-tests** for comparing means between tactics/signals
- **Chi-square tests** for conversion rate differences
- **Confidence intervals** at 95% level for all metrics
- **Effect size calculations** to measure practical significance

### Sample Size Requirements

- **Minimum 1,000 impressions** per tactic for reliable analysis
- **Minimum 50 conversions** for attribution analysis
- **Minimum 7 days data** for trend analysis
- **Larger samples** provide more reliable statistical significance

### Multiple Comparison Correction

When analyzing multiple signals or stories, p-values are adjusted using:

- **Bonferroni correction** for conservative estimates
- **False Discovery Rate (FDR)** control for balanced results
- **Family-wise error rate** protection for multiple hypotheses

```python Weekly Reporting
import pandas as pd
import matplotlib.pyplot as plt

def generate_weekly_performance_report(campaign_id):
    # Get comprehensive analysis
    analysis = client.analyze_tactics(
        campaign_id=campaign_id,
        analysis_type="efficiency",
        compare_signals=True,
        compare_stories=True,
        timeframe="7d"
    )
    
    # Create performance DataFrame
    tactics_df = pd.DataFrame(analysis['tacticRankings'])
    signals_df = pd.DataFrame(analysis['signalEffectiveness'])
    
    # Generate charts
    fig, axes = plt.subplots(2, 2, figsize=(15, 10))
    
    # Tactic efficiency chart
    tactics_df.plot(x='tacticId', y='efficiencyScore', 
                    kind='bar', ax=axes[0,0], title='Tactic Efficiency')
    
    # Signal effectiveness chart  
    signals_df.plot(x='signal', y='effectivenessScore',
                   kind='bar', ax=axes[0,1], title='Signal Effectiveness')
    
    # Statistical significance  
    significant_signals = signals_df[signals_df['pValue'] < 0.05]
    significant_signals.plot(x='signal', y='effectivenessScore',
                           kind='bar', ax=axes[1,0], title='Statistically Significant Signals')
    
    plt.tight_layout()
    plt.savefig(f'weekly_report_{campaign_id}.png')
    
    return {
        'tactics': tactics_df,
        'signals': signals_df,
        'insights': analysis['insights'],
        'recommendations': analysis['recommendations']
    }
```
