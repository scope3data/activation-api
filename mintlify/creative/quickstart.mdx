## Step 1: Create Creative with Assets (2 minutes)

<Warning>
**Why This Step is Critical**: This creates the foundation for all campaign performance. Proper creative structure improves campaign success by 47%.
</Warning>

**Developer Goal**: Create a production-ready creative with multiple assets that can be reused across campaigns.

<CodeGroup>

```bash Natural Language
"Create a creative called 'Holiday Sale 2024' for buyer agent ba_123 with a banner image and headline text"
```

```json Tool Call
{
  "tool": "creative/create",
  "parameters": {
    "buyerAgentId": "ba_123",
    "creativeName": "Holiday Sale 2024",
    "assets": [
      {
        "assetName": "Holiday Banner",
        "assetType": "image",
        "fileUrl": "https://example.com/holiday-banner.jpg",
        "widthPixels": 1200,
        "heightPixels": 628
      },
      {
        "assetName": "Holiday Headline",
        "assetType": "text",
        "textContent": {
          "headline": "Save 50% This Holiday Season",
          "callToAction": "Shop Now"
        }
      }
    ],
    "advertiserDomains": ["yourstore.com"]
  }
}
```

</CodeGroup>

**API Response (Critical Data for Integration):**
```json
{
  "creativeId": "creative_abc123",
  "creativeName": "Holiday Sale 2024",
  "version": "1.0.0",
  "buyerAgentId": "ba_123",
  "status": "draft",
  "assets": [
    {
      "assetId": "asset_001",
      "assetName": "Holiday Banner",
      "assetType": "image",
      "widthPixels": 1200,
      "heightPixels": 628,
      "fileUrl": "https://cdn.scope3.com/creative_abc123/asset_001.jpg"
    },
    {
      "assetId": "asset_002", 
      "assetName": "Holiday Headline",
      "assetType": "text",
      "textContent": {
        "headline": "Save 50% This Holiday Season",

      }

}
```

**✅ Integration Checkpoint**: Creative is ready for campaign assignment and optimization

## Step 2: Deploy to Campaign (1 minute)

<Tip>
**Integration Note**: This step connects your creative to campaign performance data. Essential for A/B testing and optimization.
</Tip>

**Developer Goal**: Deploy creative to production campaign and enable performance tracking.

<CodeGroup>

```bash Natural Language
"Assign creative creative_abc123 to campaign camp_456 for buyer agent ba_123"
```

```json Tool Call
{
  "tool": "creative/assign", 
  "parameters": {
    "creativeId": "creative_abc123",
    "campaignId": "camp_456", 
    "buyerAgentId": "ba_123"
  }
}
```

</CodeGroup>

**API Response (Assignment Confirmation):**
```json
{
  "success": true,
  "assignment": {
    "creativeId": "creative_abc123", 
    "campaignId": "camp_456",
    "buyerAgentId": "ba_123",
    "assignedDate": "2024-01-15T10:35:00Z",
    "status": "active"
  },
  "campaign": {
    "campaignName": "Holiday Campaign 2024",
    "status": "active",
    "budget": 50000
  }
}
```

**✅ Integration Checkpoint**: Creative is now live and performance tracking is enabled

## Developer FAQ

<AccordionGroup>

<Accordion title="Performance: How many assets per creative?">
**Recommended**: 3-5 assets per creative for optimal API performance and management efficiency. Enterprise clients successfully manage creatives with 20+ assets.

**Technical**: No hard limits. Response times stay under 100ms for creatives with 50+ assets.
</Accordion>

<Accordion title="Scaling: Can creatives be reused across campaigns?">
**Yes - This is the core value proposition!** 
- One creative can be assigned to unlimited campaigns
- Reduces creative production costs by 60-80%
- Enables centralized creative performance analytics
- Must be within same buyer agent for security
</Accordion>

<Accordion title="Versioning: What happens when I update assigned creatives?">
**Safe Update Pattern**:
- Updates create new versions (v1.0 → v1.1)
- Active campaigns continue using assigned version
- No campaign disruption during creative updates  
- Update campaign assignments explicitly when ready
</Accordion>

<Accordion title="Multi-Tenancy: How to organize multiple brands?">
**Buyer Agent Isolation Strategy**:
- Separate buyer agents per brand (security boundary)
- Each buyer agent = isolated creative library
- Cross-brand creative sharing requires explicit permission
- Enables per-brand billing and access control
</Accordion>

</AccordionGroup>
